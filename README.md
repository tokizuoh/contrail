## Before Development

### Git Hooks
Add the issue number to the beginning of the commit message.

```sh
$ make set-git-hooks-commit-msg
```

### Mint

```sh
# https://github.com/yonaskolb/Mint
$ git clone https://github.com/yonaskolb/Mint.git
$ cd Mint
$ swift run mint
```
